How To Set Up Maxpay Payments To Ecwid Store

Steps To Install Maxpay Payment Solution To Ecwid Store

  1. Register at Merchant Portal.
  2. Create a payment page Source: my.maxpay.com > Payment pages > Create new.
  3. Fill the required URLs.
    4. FIELD URL VALUE DESCRIPTION
    Website url https://my_store.com Shopify's shop URL
    Test success url https://ecwid.maxpay.com/success/ Test redirect url for successful payments
    Success url https://ecwid.maxpay.com/success/ Live redirect url for successful payments
    Test decline url https://ecwid.maxpay.com/error/ Test redirect url for declined payments
    Decline url https://ecwid.maxpay.com/error/ Live redirect url for declined payments
    Test callback url https://ecwid.maxpay.com/callback/ Test callback url for receiving the final response with status by Maxpay
    Callback url https://ecwid.maxpay.com/callback/ Live callback url for receiving the final response with status by Maxpay
  4. Activate a payment method/methods at Payment methods section.
  5. Create and design a payment form at Payment froms section.
  6. Set up the Maxpay's payment application from the Ecwid's store.
  7. Enter the credentials from Source: my.maxpay.com > Payment pages > Create new > API keys section.
  8. Test the payment flow on your website.
